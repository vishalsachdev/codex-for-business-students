FAQ

Common questions.

Do I need to know how to code?

No. Zero coding required. The entire course is designed for people who have never written a line of code. You'll direct AI using plain English — the same way you'd brief a contractor or manage a team.

Is Codex free?

Yes! Codex CLI works on the free ChatGPT plan. No paid subscription needed. No API keys to configure for basic usage.

How long does the course take?

About 4–5 hours total, split across four modules. Go at your own pace — do it in one sitting or spread it across a week.

What do I get at the end?

A deployed web app you can share via URL, a GitHub portfolio showcasing your work, and a real case study for your resume. Not a certificate — actual proof you can build things.

Can I use this for my own projects?

That's literally Module 3. The capstone is designed for you to build your own idea — whether it's a startup MVP, an internal tool, or a side project.
